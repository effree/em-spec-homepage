# EM-Spec Homepage

The official homepage for EM-Spec, a real-time audio spectrogram application.

## Features
- Real-time audio visualization
- Customizable spectrogram settings
- Cross-platform support (Windows & macOS)

## Development
This is a static site built with HTML and Vanilla CSS.

### Local Development
Simply open `index.html` in your browser or use a local live server.
